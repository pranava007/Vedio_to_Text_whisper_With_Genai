## How to Use

### Opening the Presentation
1. Simply open `devops-presentation.html` in any modern web browser
2. The presentation will start on the title slide

### Navigation

#### Mouse/Touch
- Click "Previous" or "Next" buttons at the bottom
- Click on the slide indicator dots (top right) to jump to specific slides
- Swipe left/right on touch devices

#### Keyboard Shortcuts
- **‚Üí / ‚Üì / Space** - Next slide
- **‚Üê / ‚Üë** - Previous slide
- **Home** - First slide
- **End** - Last slide
- **F** - Toggle fullscreen mode
- **H or ?** - Show help overlay with all shortcuts
- **Esc** - Close help overlay

### Features
- **Progress Bar**: Top of screen shows your progress through the presentation
- **Slide Counter**: Bottom center shows current slide number
- **Slide Indicators**: Top right shows dots for each slide
- **Auto-save**: Your current position is saved and restored when you reopen

### Presenting
1. Press **F** to enter fullscreen mode for the best presentation experience
2. Use arrow keys or space bar to navigate through slides
3. The presentation is fully self-contained - no internet connection required after initial load

### Printing/Exporting
- Use your browser's print function (Ctrl+P / Cmd+P) to print or save as PDF
- All slides will be formatted for printing automatically

## Customization

### Editing Content
- Open `devops-presentation.html` in a text editor
- Each slide is contained in a `<div class="slide">` element
- Modify text, add/remove slides as needed

### Changing Colors
- Edit `presentation-styles.css`
- Main gradient colors: `#667eea`, `#764ba2`, `#f093fb`
- Background: `#0f0c29`, `#302b63`, `#24243e`

### Adding Slides
1. Copy an existing slide `<div class="slide">` block
2. Update the `data-slide` attribute with the next number
3. Modify the content
4. The navigation will automatically update
